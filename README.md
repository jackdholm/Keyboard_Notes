# Keyboard_Notes
Entirely keyboard-based note-taking program

## Controls
New Tab: **Ctrl-N**
Create Note Mode: **Enter**
Navigation Mode: Escape
Tab Navigation: H/L
Note Navigation: J/K
Delete Note: Delete
Delete Tab: Shift+Delete
Change Order: Ctrl-O
##### When Creating notes
Title Text: #T
Add Note: Shift+Enter
Shift+H/J/K/L: Navigation
